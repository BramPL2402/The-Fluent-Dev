# Curso de ITI

¡Este es nuestro repositorio de aprendizaje! Este espacio está diseñado para documentar tu evolución en la carrera. Aquí alojarás todos los retos, prácticas y proyectos que desarrolles durante las distintas fases del curso.

---

## Hoja de Ruta (Roadmap)

El curso está dividido en 4 pilares fundamentales.

### Fase 1: Lógica y Algoritmia
*Fundamentos de pensamiento computacional.*
- [ ] Estructuras de control (If, For, While).
- [ ] Recursividad y optimización.
- [ ] Análisis de complejidad (Notación Big O).

### Fase 2: Programación (Python/C++)
*Dominio del lenguaje y sintaxis.*
- [ ] Manejo de tipos de datos y estructuras.
- [ ] Programación Orientada a Objetos (POO).
- [ ] Manipulación de archivos.

### Fase 3: Bases de Datos
*Gestión de información estructurada.*
- [ ] Diseño de Entidad-Relación.
- [ ] Consultas SQL (Select, Joins, Aggregations).
- [ ] Normalización.

### Fase 4: Redes y Ciberseguridad
*Infraestructura y protección.*
- [ ] Protocolos de red y modelo OSI.
- [ ] Pentesting básico y auditoría.
- [ ] Seguridad en aplicaciones.
